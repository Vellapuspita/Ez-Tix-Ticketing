# Ez-Tix-Ticketing

Ez-Tix adalah platform yang dirancang untuk memberikan kemudahan dalam manajemen dan pemesanan tiket digital (Easy Ticketing). Produk ini hadir sebagai solusi bagi penyelenggara acara untuk mengelola distribusi tiket, sistem pembayaran, hingga validasi pengunjung secara otomatis. Dengan Ez-Tix, pengalaman menyelenggarakan dan menghadiri acara menjadi lebih seamless, modern, dan efisien.

---

## Tech Stack
- **Frontend:** React (Vite)
- **Backend:** Node.js & Express.js
- **Database:** MongoDB dengan Mongoose
